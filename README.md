В проекте 7 спринта мною реализовано тестирование API учебного сервиса "Яндекс Самокат".

Структура проекта:

allure_results - Allure-отчёт
tests - содержит тестовые сценарии
curl.py - содержит URL-адреса используемых страниц
data.py - содержит тестовые данные
helper.py - содержит вспомогательные функции для генерации данных
requirements.txt - файл с внешними зависимостями

Папка tests содержит:
conftest.py - содержит фикстуры
test_create_courier.py - тесты на создание курьера
test_get_order_list.py - тесты на получение списка заказов
test_login.py - тесты на логин курьера
test_new_order.py - тесты на создание заказа
